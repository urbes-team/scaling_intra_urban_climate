# intra_urban_climate_scaling

Repository for the results of the paper **“Scaling intra-urban climate fluctuations”** (arXiv: https://arxiv.org/abs/2505.19998).

## Structure

- `src/`             – Main code modules for data processing, analysis, and plotting  
- `datasets/`  – Generated CSV datasets used in the analysis  
- `requirements.txt` – Python dependencies  
- `README.md`        – This file  

## Installation

Install all required Python packages with:

    pip install -r requirements.txt
